# Primary Use Case Figure Workflow

This directory contains the reproducible workflow scaffold for the manuscript figure:

`CLONEID links event history, growth dynamics and karyotypic state in a longitudinal adaptation experiment.`

Current step status:

- Step 1 complete: the subtree `.RDS` input was inspected and mapped to the planned panels.
- Step 2 complete after this scaffold: workflow layout, `.Rmd`, and helper stubs are in place.
- Step 3 complete: panel `A` and `D/E` manifests are auto-derived from the subtree bundle.
- Step 4 in progress: panels `B` and `C1-C4` are implemented from the subtree `.RDS`.
- Step 5 in progress: panels `D/E` are implemented against the downloaded `GenomePerspective` export payload; panel `A` is now wired to render directly from `panelA.zip`.
